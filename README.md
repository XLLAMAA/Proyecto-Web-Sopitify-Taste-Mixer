# Proyecto-Web-Sopitify-Taste-Mixer
Aplicación web “Spotify Taste Mixer” hecha con Next.js. Permite iniciar sesión con Spotify, elegir preferencias mediante widgets (artistas, canciones, géneros, décadas, estado de ánimo, popularidad) y generar una playlist personalizada que luego se puede modificar, recargar y marcar canciones como favoritas.
